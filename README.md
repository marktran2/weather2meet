# weather2meet

### Project Idea
Meeting with others is a hassle. Especially on a rainy day. Don't want to miss out on the perfect weather to meetup and have some fun? **weather2meet** is for you!

### Technicalities
Programming languages this project uses are:
**Frontend**:
* ReactJS
* TailwindCSS

**Backend**:
* NodeJS
* ExpressJS
